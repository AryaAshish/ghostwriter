Ghostwriter - Central Planning Document

Product Vision
Mission: Build an AI content generation tool for Instagram and YouTube creators in India that mimics their exact personal tone, identifies trending topics, and writes high-performing video scripts tailored to their niche and voice.

Core Value Proposition: "This sounds like I wrote it — but I didn't have to."

MVP Goals (Phase 1 - 6 weeks)
User Acquisition: 10–15 paying creators
Personalization: Q&A-based onboarding for tone capture
Content Delivery: 30 scripts/month per user
Monetization: ₹2,999/month with ₹999 trial
Marketing: Instagram-first content strategy

Product Architecture

Core Components
1. Instagram Marketing Page - Brand awareness and user acquisition
2. Onboarding Flow - Niche and voice style collection
3. Script Generation Engine - GPT-powered, prompt-tuned system
4. Trend Integration - Manual daily topic injection (MVP)
5. Delivery System - Multi-channel (Email/WhatsApp/Google Docs)
6. Payment Layer - Razorpay/Stripe integration
7. Creator Dashboard - Script management (stretch goal)

Tech Stack
Component | Technology
Frontend | Next.js / React / Vercel
Backend | Node.js / Supabase / Firebase
AI Layer | GPT-4 Turbo via OpenAI API
Database | Supabase/Postgres
Delivery | WhatsApp (Twilio), Email, Google Docs
Payments | Razorpay / Stripe
Version Control | GitHub + Linear/Windsurf

Epic Overview

Epic 1: Instagram Marketing & Acquisition
Timeline: Week 1-2
Priority: High
Goal: Build early audience and warm up creator market
Instagram page creation and branding
Daily content publishing (1-2 posts)
Engagement tracking and lead generation
Viral content strategy testing

Epic 2: Script Personalization Onboarding
Timeline: Week 2-3
Priority: High
Goal: Collect data to personalize script outputs
5-6 style assessment questions
Niche selection interface
Optional content sample upload
Secure profile data storage

Epic 3: Script Generation Engine
Timeline: Week 2-4
Priority: High
Goal: Generate niche + tone-aligned scripts daily
30 scripts/month delivery system
Prompt templates per niche with tone variations
Manual trend integration workflow
20 edit/rewrite options per script

Epic 4: Delivery Mechanism
Timeline: Week 3-4
Priority: High
Goal: Seamlessly deliver personalized scripts
Multi-channel delivery (WhatsApp, Email, Google Docs)
Batch delivery system with queue management
Script rewrite request handling
Content archive access

Epic 5: Payments & Trial Flow
Timeline: Week 4-5
Priority: High
Goal: Monetize through lean checkout
₹2,999/month subscription with ₹999 trial
UPI + card payments via Razorpay/Stripe
Subscription management and billing
Payment analytics and failed payment handling

Epic 6: Creator Dashboard (Stretch Goal)
Timeline: Week 5-6+
Priority: Low
Goal: Enhanced user experience and retention
Script library management
Feedback system for AI improvement
In-dashboard editing capabilities
Performance analytics integration
